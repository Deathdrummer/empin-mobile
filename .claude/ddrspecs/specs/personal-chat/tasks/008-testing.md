# Task: Тестирование и проверка качества

## Goal
Протестировать все функции личного чата и исправить возможные ошибки.

## Files
- Все созданные/измененные файлы

## Steps
1. **Тестирование создания чата:**
   - Открыть ChatsTab
   - Кликнуть на сотрудника
   - Проверить создание чата в БД
   - Проверить корректность нормализации участников

2. **Тестирование отправки сообщений:**
   - Отправить текстовое сообщение
   - Проверить сохранение в БД
   - Проверить отображение на экране
   - Проверить разделение "свои/чужие"

3. **Тестирование медиа:**
   - Прикрепить фото из галереи
   - Сделать фото через камеру
   - Снять видео
   - Прикрепить документ
   - Проверить отображение каждого типа
   - Проверить корректное сохранение в storage

4. **Тестирование аудио:**
   - Записать голосовое сообщение
   - Проверить отправку
   - Проверить воспроизведение

5. **Тестирование реакций:**
   - Добавить реакцию на сообщение
   - Проверить отображение счетчика
   - Удалить реакцию
   - Проверить исчезновение реакции

6. **Тестирование ответов:**
   - Ответить на сообщение
   - Проверить отображение цитаты
   - Проверить связь в БД

7. **Тестирование редактирования:**
   - Двойной тап на своем сообщении
   - Изменить текст
   - Проверить отметку "изменено"
   - Проверить корректное время редактирования

8. **Тестирование удаления:**
   - Долгое нажатие на сообщение
   - Выбрать "Удалить"
   - Проверить удаление из БД
   - Проверить удаление медиафайлов из storage

9. **Тестирование прав доступа:**
   - Попробовать редактировать чужое сообщение (должна быть ошибка)
   - Проверить работу прав `mobile-app-can-edit-comment:site`
   - Проверить работу прав `mobile-app-can-delete-comment:site`

10. **Тестирование UI/UX:**
    - Проверить автоскролл к новым сообщениям
    - Проверить Pull-to-refresh
    - Проверить поведение клавиатуры на iOS
    - Проверить отображение на разных размерах экрана

11. **Тестирование ошибок:**
    - Попробовать отправить пустое сообщение (должна быть валидация)
    - Отключить интернет и проверить обработку ошибок
    - Проверить корректность Toast уведомлений

12. **Исправление найденных багов:**
    - Зафиксировать все найденные проблемы
    - Исправить их
    - Повторить тесты

## Acceptance
- [ ] Все тесты пройдены успешно
- [ ] Текстовые сообщения отправляются и отображаются
- [ ] Медиа (фото, видео) работают корректно
- [ ] Аудио записывается и воспроизводится
- [ ] Документы прикрепляются и скачиваются
- [ ] Реакции добавляются и удаляются
- [ ] Ответы на сообщения работают
- [ ] Редактирование сохраняется
- [ ] Удаление убирает сообщение и медиа
- [ ] Права доступа работают корректно
- [ ] UI/UX соответствует групповому чату
- [ ] Ошибки обрабатываются корректно
- [ ] Нет критических багов
