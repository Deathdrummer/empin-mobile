# Кнопка "Аккаунт" в нижнем меню с всплывающим меню

## Цель
Заменить кнопку "Выйти" в нижнем меню на кнопку-иконку "Аккаунт" с силуэтом пользователя, при тапе на которую раскрывается вверх меню с пунктами: План-график работ, Мессенджер, Выйти.

## Контекст
Текущее нижнее меню имеет 4 кнопки: Календарь, Фильтр, Пустая секция, Выйти.
Нужно заменить кнопку "Выйти" на кнопку "Аккаунт", которая открывает меню для навигации между разделами приложения.

## Выбранное решение
- **Подход**: ActionSheet меню (slide from bottom)
- **Библиотека**: @expo/react-native-action-sheet (уже установлен в проекте)
- **Навигация**: React Navigation Stack Navigator
- **Обоснование**:
  - ActionSheet уже используется в проекте (ChatSection.js)
  - Минимум зависимостей (0kb дополнительно)
  - React Navigation уже установлен (@react-navigation/native 7.1.18)
  - Стандартное поведение для мобильных приложений

## Связанные файлы
- `src/components/BottomMenu.js` - компонент нижнего меню
- `src/screens/Timesheet/index.js` - главный экран (План-график работ)
- `App.js` - корневой компонент приложения
- Новый: `src/screens/MessengerScreen.js` - экран Мессенджера (заглушка)

## Ограничения

- Сохранить существующий функционал выхода из аккаунта
- Сохранить все существующие обработчики и логику TimesheetScreen
