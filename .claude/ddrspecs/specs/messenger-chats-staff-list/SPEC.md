# Спецификация: Список сотрудников в разделе "Чаты"

## Обзор
Реализовать отображение списка всех сотрудников в подразделе "Чаты" раздела "Мессенджер".

## Архитектура

### Компоненты
- `ChatsTab` - компонент вкладки со списком сотрудников

### API
- `GET /timesheet/staff` → `[{id, sname, fname, mname}]`

## UI/UX
- FlatList со списком сотрудников
- Аватар с первой буквой фамилии
- ФИО в формате "Фамилия Имя Отчество"
- Pull-to-refresh для обновления
- Loading индикатор при загрузке

## Файловая структура
```
src/screens/
├── MessengerScreen.js      ← интеграция ChatsTab
└── Messenger/
    └── ChatsTab.js         ← новый компонент
```
