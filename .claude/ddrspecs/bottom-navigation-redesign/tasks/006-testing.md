# Task 6: Тестирование и финальная проверка

## Цель
Протестировать все функции новой навигации, убедиться в корректной работе всех кнопок, переходов и интерактивных элементов.

## Контекст
Финальная проверка всех реализованных функций перед завершением задачи.

## Файлы
Все изменённые файлы

## План
1. [ ] **Тест левой кнопки "три точки":**
   - Открывается ActionSheet меню
   - Пункт "Выйти" корректно выходит из приложения
   - "Отмена" закрывает меню
2. [ ] **Тест правой кнопки переключения:**
   - В Timesheet показывает иконку мессенджера
   - В Messenger показывает иконку календаря
   - Переход плавный и корректный
3. [ ] **Тест раздела "План-график работ":**
   - Средние кнопки "Календарь" и "Фильтры" отображаются
   - Календарь открывается и работает
   - Фильтры открываются и работают
   - Индикатор активных фильтров отображается корректно
   - Очистка фильтров работает
4. [ ] **Тест раздела "Мессенджер":**
   - Средние кнопки "Чаты" и "История звонков" отображаются
   - Переключение между подразделами работает
   - Активная кнопка подсвечивается
   - Контент меняется при переключении
5. [ ] **Общие тесты:**
   - Навигация между разделами сохраняет состояние
   - Нет утечек памяти или перерендеров
   - Все тапы отзывчивы
   - Визуальная консистентность

## Критерии готовности
- [ ] Все функции работают без ошибок
- [ ] Нет регрессий в существующих функциях
- [ ] UI выглядит консистентно на обоих экранах
- [ ] Переходы плавные и без багов
- [ ] Код чистый, без console.log и дебаг-кода
