# Task 2: Реализация popup меню для кнопки "три точки"

## Цель
Реализовать левую кнопку с иконкой "три вертикальные точки", которая открывает ActionSheet меню с пунктом "Выйти".

## Контекст
Используем уже установленную библиотеку `@expo/react-native-action-sheet` (аналогично меню при долгом зажатии на комментарий).

## Файлы
- `src/components/BottomMenu.js`

## План
1. [x] Добавить иконку Ionicons `ellipsis-vertical` для кнопки "три точки"
2. [x] Создать handleDotsMenuPress() - вызов ActionSheet
3. [x] Настроить опции ActionSheet: ["Выйти", "Отмена"]
4. [x] Добавить иконки в опции меню (Ionicons `log-out-outline` для выхода)
5. [x] Подключить onLogout callback при выборе "Выйти"
6. [x] Обновить renderLeftButton() с реальной реализацией

## Критерии готовности
- [x] Кнопка "три точки" отображается слева
- [x] Тап на кнопку открывает ActionSheet снизу
- [x] ActionSheet содержит пункт "Выйти" с иконкой
- [x] Выбор "Выйти" вызывает onLogout
- [x] Выбор "Отмена" закрывает меню без действий

## Результат
- Изменено: `src/components/BottomMenu.js`
- Добавлен импорт `useActionSheet` из `@expo/react-native-action-sheet`
- Реализован обработчик `handleDotsMenuPress()` с настроенным ActionSheet
- ActionSheet содержит пункты "Выйти" (с иконкой log-out-outline) и "Отмена"
- При выборе "Выйти" вызывается `onLogout` callback
- Кнопка "три точки" активна и открывает меню по тапу

## Проблемы и решения

### 🐛 Проблема 1: Лишний пункт "Отмена" и большое расстояние между иконкой и текстом
**Дата**: 2026-01-19 19:15

**Описание**:
- В меню ActionSheet отображался лишний пункт "Отмена"
- Слишком большое расстояние между иконкой и названием пункта "Выйти"

**Решение**:
- Убрана "Отмена" из массива options (ActionSheet закрывается свайпом вниз)
- Удален параметр `cancelButtonIndex` из конфигурации
- Уменьшен размер иконки с 24px до 20px для улучшения компактности

**Изменено**:
- `src/components/BottomMenu.js` - обновлен `handleDotsMenuPress()`:
  - options: `['Выйти']` (было `['Выйти', 'Отмена']`)
  - icon size: `20` (было `24`)
  - удален `cancelButtonIndex`

**Источники**:
- [GitHub Issue #174](https://github.com/expo/react-native-action-sheet/issues/174) — проблемы с выравниванием иконок и текста
