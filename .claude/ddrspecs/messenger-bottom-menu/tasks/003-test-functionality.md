# Task 3: Тестирование функционала

## Цель
Проверить корректность работы нижнего меню на обоих экранах (Timesheet и Messenger).

## Контекст
Нужно убедиться что изменения не сломали существующий функционал и новое меню работает корректно.

## Файлы
- Тестирование на эмуляторе BlueStacks

## План
1. [x] Запустить приложение на эмуляторе
2. [x] Проверить работу меню в разделе "План-график работ":
   - Все 4 кнопки отображаются
   - Кнопка "Календарь" открывает календарь
   - Кнопка "Фильтр" открывает фильтры
   - Кнопка "Аккаунт" → action sheet → "Мессенджер" переходит в Мессенджер
3. [x] Проверить работу меню в разделе "Мессенджер":
   - Видны 3 пустые позиции и кнопка "Аккаунт"
   - Кнопка "Аккаунт" → action sheet → "План-график работ" возвращает на Timesheet
4. [x] Проверить визуальную консистентность:
   - Меню одинаковой высоты на обоих экранах
   - Кнопка "Аккаунт" на одинаковой позиции
   - Отступы и стили идентичны

## Критерии готовности
- [x] Меню в "План-график работ" работает без изменений
- [x] Меню в "Мессенджер" отображается с 3 пустыми позициями и кнопкой "Аккаунт"
- [x] Навигация между экранами работает в обе стороны через action sheet
- [x] Визуально меню выглядит консистентно на обоих экранах
- [x] Нет ошибок в консоли
