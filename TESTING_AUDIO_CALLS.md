# Инструкции по тестированию аудиозвонков (LiveKit Cloud)

## Подготовка к тестированию

### 1. Настройка LiveKit Cloud

1. Зарегистрируйтесь на https://cloud.livekit.io/
2. Создайте новый проект
3. В Project Settings получите:
   - `API Key`
   - `API Secret`
   - `WebSocket URL` (вида `wss://your-project.livekit.cloud`)
4. Добавьте в backend `.env`:
   ```env
   LIVEKIT_API_KEY=your_api_key
   LIVEKIT_API_SECRET=your_api_secret
   LIVEKIT_URL=wss://your-project.livekit.cloud
   ```

### 2. Сборка приложения

После изменения зависимостей обязательно выполнить:

```bash
npm install
npx expo prebuild --clean
npx expo run:android
```

### 3. Настройка тестовых устройств

- 2 физических устройства ИЛИ
- 1 физическое устройство + 1 эмулятор BlueStacks
- Оба подключены к интернету

### 4. Создание тестовых пользователей

- User A и User B в системе
- Залогиниться на разных устройствах

## Тестовые сценарии

### Сценарий 1: Исходящий звонок (User A → User B)

**Шаги:**
1. User A: Открыть мессенджер
2. User A: Нажать кнопку звонка рядом с User B в списке контактов
3. ✅ Проверить: CallModal открылся с статусом "Звоним..."
4. User B: Нажать "Принять" в CallModal
5. ✅ Проверить: Оба видят статус "Разговор"
6. User A: Говорить → ✅ User B слышит
7. User B: Говорить → ✅ User A слышит
8. ✅ Проверить: Таймер длительности увеличивается
9. User A: Нажать "Завершить"
10. ✅ Проверить: CallModal закрылся у обоих
11. ✅ Проверить в БД: `messenger_calls` запись со status='completed'

**Результат:**
- [ ] Звонок инициируется
- [ ] Звонок принят
- [ ] Звук передаётся в обе стороны
- [ ] Таймер работает
- [ ] Звонок завершён
- [ ] Данные сохранены в БД

### Сценарий 3: Управление звуком (Mute/Speaker)

**Шаги:**
1. User A и User B в активном звонке
2. User A: Нажать "Выкл. микрофон" (Mute)
3. User A: Говорить → ✅ User B НЕ слышит
4. User A: Нажать "Вкл. микрофон" (Unmute)
5. User A: Говорить → ✅ User B слышит
6. User B: Нажать "Динамик"
7. ✅ Проверить: Звук переключился на динамик
8. User B: Нажать "Наушники"
9. ✅ Проверить: Звук переключился обратно

**Результат:**
- [ ] Mute/Unmute работает
- [ ] Динамик/Наушники переключается
- [ ] UI кнопок обновляется (визуальное состояние)

2. Попробовать позвонить
3. ✅ Проверить: Alert "Разрешите доступ к микрофону"

- [ ] Обработка нет сети

## Известные ограничения

1. **Push-уведомления**: Не входят в scope текущей задачи, будут реализованы отдельно
2. **LiveKit Cloud**: Требуется настройка API ключей на backend (`.env`)
3. **Сборка**: После `npm install` необходим `npx expo prebuild --clean`

## Логи для отладки

Проверять логи в консоли:
```

Participant joined: [identity]
Participant left: [identity]
```
