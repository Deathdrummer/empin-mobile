# Краткая выжимка запроса

**Задача:** Унификация формата отображения имен сотрудников в разделе "Мессенджер"

**Проблема:**
- В разделе "План-график работ" имя бригады (мастера) выводится в формате "Фамилия И.О."
- В разделе "Мессенджер -> Чаты" и в чате с конкретным пользователем имена выводятся в полном формате "Фамилия Имя Отчество"

**Решение:**
Использовать в мессенджере тот же формат, что и в план-графике работ: "Фамилия И.О."

**Затронутые компоненты:**
- `src/screens/Messenger/ChatsTab.js` - список чатов
- `src/screens/Messenger/ChatScreen.js` - экран чата с конкретным пользователем
