# Архив завершенных задач DDRSpecs

Папка содержит завершенные и смерженные задачи для истории проекта.

## Статус задач в архиве

Все задачи имеют статус `done` и смержены в `main`.

## Доступ к истории

Краткий контекст выполненных задач доступен в [HISTORY.md](../HISTORY.md).

## Восстановление

Если нужно вернуть задачу из архива:
```bash
cp -r .claude/ddrspecs/archive/{task-slug} .claude/ddrspecs/
```
