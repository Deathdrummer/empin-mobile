# Краткое описание запроса

Реализовать полнофункциональный личный чат в разделе "Мессенджер -> Чаты" для общения между двумя пользователями.

## Контекст
- Сейчас в разделе мессенджера есть заглушка ChatScreen
- Существует работающая реализация группового чата в "План-график работ -> Бригада -> Объект -> Чат"
- Необходимо использовать весь функционал группового чата для личного чата

## Ключевые требования
- Личный чат должен работать идентично групповому чату
- Поддержка всех возможностей: текст, медиа (фото/видео), аудио, документы, реакции, ответы, редактирование
- Отдельная база данных для личных чатов (с возможностью расширения до групповых чатов в будущем)
