# Task 2: Реализация popup меню для кнопки "три точки"

## Цель
Реализовать левую кнопку с иконкой "три вертикальные точки", которая открывает ActionSheet меню с пунктом "Выйти".

## Контекст
Используем уже установленную библиотеку `@expo/react-native-action-sheet` (аналогично меню при долгом зажатии на комментарий).

## Файлы
- `src/components/BottomMenu.js`

## План
1. [ ] Добавить иконку Ionicons `ellipsis-vertical` для кнопки "три точки"
2. [ ] Создать handleDotsMenuPress() - вызов ActionSheet
3. [ ] Настроить опции ActionSheet: ["Выйти", "Отмена"]
4. [ ] Добавить иконки в опции меню (Ionicons `log-out-outline` для выхода)
5. [ ] Подключить onLogout callback при выборе "Выйти"
6. [ ] Обновить renderLeftButton() с реальной реализацией

## Критерии готовности
- [ ] Кнопка "три точки" отображается слева
- [ ] Тап на кнопку открывает ActionSheet снизу
- [ ] ActionSheet содержит пункт "Выйти" с иконкой
- [ ] Выбор "Выйти" вызывает onLogout
- [ ] Выбор "Отмена" закрывает меню без действий
