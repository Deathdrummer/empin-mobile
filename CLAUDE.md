## DDRSpecs Task Management

Проект использует DDRSpecs микрофреймворк для управления задачами.

**Основные команды:**
- `/ddrspec:new {описание}` — Создать новую задачу
- `/ddrspec:run` — Продолжить выполнение активной задачи
- `/ddrspec:list` — Показать список всех задач
- `/ddrspec:bug {описание}` — Решить проблему в текущей задаче
- `/ddrspec:commit` — Закоммитить изменения выполненных задач
- `/ddrspec:done` — Завершить активную задачу и merge в main

**Структура:**
```
.claude/ddrspecs/
├── AGENTS.md         # Инструкции для агента
├── ACTIVE.md         # Указатель на активную задачу
├── HISTORY.md        # История выполненных задач
└── {task-slug}/      # Папки задач
```

**Документация:** `.claude/ddrspecs/AGENTS.md`
