# Спецификация: Унификация формата имен в мессенджере

## Цель

Привести формат отображения имен сотрудников в разделе "Мессенджер" к единому стандарту "Фамилия И.О.", используемому в разделе "План-график работ".

## Требования

### Функциональные требования

1. **Список чатов** (`ChatsTab.js`):
   - Имена сотрудников отображаются в формате "Фамилия И.О."
   - Используется утилита `formatShortName()` из `src/utils/formatName.js`

2. **Экран чата** (`ChatScreen.js`):
   - Имя собеседника в заголовке отображается в формате "Фамилия И.О."
   - Имя передается через `route.params.staffName` уже в правильном формате

### Технические требования

1. **Использовать существующую утилиту:**
   - Импортировать `formatShortName` из `src/utils/formatName.js`
   - Удалить локальную функцию `formatName` из `ChatsTab.js`

2. **Обратная совместимость:**
   - Утилита `formatShortName` уже используется в `TeamCard.js`
   - Никаких изменений в существующей логике не требуется

## Затронутые файлы

- ✏️ `src/screens/Messenger/ChatsTab.js` - изменить функцию форматирования имени
- ℹ️ `src/screens/Messenger/ChatScreen.js` - без изменений (получает готовое имя)
- ℹ️ `src/utils/formatName.js` - без изменений (утилита уже существует)

## Acceptance Criteria

- [ ] В списке чатов имена отображаются в формате "Фамилия И.О."
- [ ] В экране чата имя собеседника в заголовке отображается в формате "Фамилия И.О."
- [ ] Используется утилита `formatShortName()` вместо локальной функции
- [ ] Код соответствует существующему стилю проекта
- [ ] Нет регрессий в отображении имен
