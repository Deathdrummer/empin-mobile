# Полная переделка навигации (нижняя панель меню)

## Цель
Переработать компонент BottomMenu для создания унифицированной навигации с фиксированными крайними кнопками и динамическими средними кнопками в зависимости от активного раздела.

## Контекст
Текущая реализация использует условные пропсы `showCalendar`/`showFilter` для переключения контента меню. Новая концепция требует:
- **Левая кнопка** (фиксированная): три вертикальные точки → popup меню с пунктом "Выйти"
- **Правая кнопка** (фиксированная): динамическая иконка перехода в другой раздел
- **Средние кнопки** (динамические):
  - **План-график работ**: Календарь, Фильтры
  - **Мессенджер**: Чаты, История звонков

## Выбранное решение
- **Popup меню**: ActionSheet (уже используется в проекте `@expo/react-native-action-sheet`)
- **Навигация**: React Navigation (плавные переходы через `navigation.replace()`)
- **Структура**: 4 позиции с фиксированными крайними и динамическими средними кнопками

## Связанные файлы
- `src/components/BottomMenu.js` - основной компонент нижнего меню
- `src/screens/MessengerScreen.js` - экран мессенджера
- `src/screens/Timesheet/index.js` - экран плана-графика работ
- `App.js` - конфигурация навигации

## Ограничения
- Сохранить визуальную консистентность
- Использовать существующие библиотеки (не добавлять новые зависимости)
- Сохранить работоспособность всех существующих функций (выход, календарь, фильтры, подразделы мессенджера)
