### 2026-01-19: Полная переделка навигации (нижняя панель меню)
- **Slug**: bottom-navigation-redesign
- **Branch**: feature/bottom-navigation-redesign (merged)
- **Описание**: Переработан компонент BottomMenu для унифицированной навигации с фиксированными крайними кнопками (три точки слева, динамическая кнопка переключения раздела справа) и динамическими средними кнопками в зависимости от активного раздела.
- **Ключевые решения**:
  - ActionSheet для popup меню выхода
  - React Navigation с replace() для плавных переходов
  - Структура: 4 позиции с динамическим контентом
  - Патч библиотеки @expo/react-native-action-sheet (marginRight: 4) для корректного отображения иконок

### 2026-01-17: Кнопка "Аккаунт" в нижнем меню с всплывающим меню
- **Slug**: account-menu-bottom-bar
- **Branch**: feature/account-menu-bottom-bar (merged)
- **Описание**: Заменена кнопка "Выйти" на кнопку-иконку "Аккаунт" с силуэтом пользователя. При тапе открывается ActionSheet меню с пунктами: План-график работ, Мессенджер, Выйти.
- **Ключевые решения**:
  - ActionSheet меню (@expo/react-native-action-sheet)
  - React Navigation Stack Navigator
  - Создан экран MessengerScreen (заглушка)
